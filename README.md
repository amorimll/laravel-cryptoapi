## Tecnologias necessárias

* PHP 8
* Composer

## Instalação local

Use o comando "git clone https://github.com/amorimll/laravel-cryptoapi" em uma pasta vazia, em seguida abra a pasta que foi criada, terá o nome de "laravel-cryptoapi", abra um terminal e execute o comando "composer install" para instalar as depedências, para configurar o banco de dados, abra o arquivo .env.example, mude o nome para .env, e troque as informações pelo seu banco de dados correspondente, após a configuração do banco de dados, execute o comando "php artisan migrate"
